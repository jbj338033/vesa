# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please send an email to the maintainer via their GitHub profile contact.

### What to expect

- **Acknowledgement** within 48 hours
- **Status update** within 7 days
- We will work with you to understand and resolve the issue before any public disclosure

### Scope

This project handles keyboard and mouse input over the network. Security-relevant areas include:

- QUIC transport and TLS certificate handling
- Input capture and injection permissions
- Network protocol parsing
